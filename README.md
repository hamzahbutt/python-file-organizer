# Python File Organizer

A simple Python script that automatically organizes files into folders based on their file extensions.

## Features

- Organizes images
- Organizes documents
- Organizes videos
- Organizes audio
- Creates folders automatically
- Writes a log file of moved files

## Technologies Used

- Python
- os
- shutil

## How to Run

1. Run the script.

```bash
python organizer.py
```

2. Enter the folder path when prompted.

## Supported File Types

| Extension | Folder |
|-----------|--------|
| .jpg .jpeg .png .gif | Images |
| .pdf .docx .txt | Documents |
| .mp4 .mov | Videos |
| .mp3 .wav | Audios |
| .zip | Archives |
